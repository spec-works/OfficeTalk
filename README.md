# OfficeTalk
[![Registry](https://img.shields.io/badge/Registry-SpecWorks-blue)](https://spec-works.github.io/registry/parts/officetalk/)

Implementation library for the `application/officetalk` media type — deterministic modifications to Microsoft Office documents.

## Specifications

Implements [OfficeTalk 1.0](specification.md) and [Office Open XML (ECMA-376)](https://www.ecma-international.org/publications-and-standards/standards/ecma-376/).

See [specs.json](specs.json) for complete specification references.
